# पश्चिम की यात्रा विश्वकोश · Journeypedia

**[journeypedia.org](https://journeypedia.org)** *पश्चिम की यात्रा* (西游记) पर एक बहुभाषी विश्वकोश है, जो 11 भाषाओं में पात्रों, राक्षसों, शस्त्रों, दिव्य शक्तियों, स्थानों और मूल कथा को कवर करता है। प्रत्येक प्रविष्टि मूल पाठ का गहन विश्लेषण प्रस्तुत करती है — पात्रों की पृष्ठभूमि, महत्वपूर्ण प्रसंग, प्रतीकात्मक अर्थ और सांस्कृतिक संदर्भ।

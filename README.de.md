# Reise nach Westen Enzyklopädie · Journeypedia

**[journeypedia.org](https://journeypedia.org)** ist eine mehrsprachige Enzyklopädie über *Die Reise nach Westen* (西游记), die Charaktere, Dämonen, Artefakte, Fähigkeiten, Orte und die Originalgeschichte in 11 Sprachen abdeckt. Jeder Eintrag bietet eine tiefgehende Analyse des Quelltextes — Hintergründe der Figuren, Schlüsselszenen, symbolische Bedeutungen und kulturellen Kontext.

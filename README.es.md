# Enciclopedia de Viaje al Oeste · Journeypedia

**[journeypedia.org](https://journeypedia.org)** es una enciclopedia multilingüe sobre *Viaje al Oeste* (西游记), que cubre personajes, demonios, artefactos, poderes, lugares y la historia original en 11 idiomas. Cada entrada ofrece un análisis profundo del texto fuente — trasfondos de personajes, momentos clave, significados simbólicos y contexto cultural.
